# Node.js-Express-boilerplate
A boilerplate for building production-ready RESTful APIs using Node.js, Express, Typescript, and PostgreSQL

# Architecture

```text
├─ test           // All the testing code in it
├─ dist          // Build version
├─ src
│  ├─ config        
│  ├─ controllers   
│  ├─ middlewares   
│  ├─ routes     
│  ├─ services        
│  ├─ utils        
```
